# 《大结果》— 3 分钟拿捏一个富哥

一款高反馈、高传播、单局约 3 分钟的**三维口味文字冒险小游戏**，手机端优先。**纯静态单文件**：整个游戏就在 `index.html` 里，零外部依赖、可离线双击即玩、也可直接发给别人。

## 玩法内核
- **人设**：A–F 六列各抽一词拼成（F 人设类型必出），六列 +/− 累加成你的 **魅力**。
- **三维（可正可负）**：魅力（土甜◄►高级）· 框架（倒贴◄►拿捏）· 务实（纯情◄►搞钱）。
- **上头值 0–100**：你的每步选择在「大哥加权口味方向」上的累计契合度（主反馈条）。
- **结局四档 18 个**：大结果（反向拿捏/结婚/送出国/关起来/买房/包养）· 小结果（铂金包/车/表/香奶奶）· 没结果·白上桌（留在YC/没上桌/白入局/陪跑）· 负结果（查无此人/倒亏/翻车社死/反被薅）。
- 每局随机人设/大哥/场地；结局给一张「看似贬低实则封神」的鉴定书，可一键导出 PNG（手机存相册 / 电脑存文件）；结局图鉴四档收集、可回看。

## 本地玩
直接双击 `index.html` 即可（无需服务器、无需联网）。

## 部署到 Cloudflare Pages（网页操作，纯静态）
1. 代码已在 GitHub：`https://github.com/kulainisi/Dajieguo`
2. Cloudflare 控制台 → **Workers & Pages → Create → 选「Pages」→ Connect to Git** → 选该仓库。
3. 构建设置：
   - Framework preset：**None**
   - Build command：**留空**
   - Build output directory：**`.`**（根目录）
4. Save and Deploy。之后每次 `git push` 自动重新部署，线上地址形如 `https://dajieguo.pages.dev`。

> 本项目是纯静态站，没有任何后端 / 构建步骤，Pages 直接把 `index.html` 发布出去即可。
