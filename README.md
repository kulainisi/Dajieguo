# 《大结果》

### 3 分钟拿捏一个富哥 · 三维口味文字冒险

**你是什么茶 × 他好哪口 × 你怎么打** —— 十轮对话，看你能拿多大的结果。

</div>

---

一款高反馈、高传播、**单局约 3 分钟**、手机端优先的文字冒险小游戏。
**纯静态单文件**：整个游戏就在一个 `index.html` 里，零外部依赖、可离线双击即玩、也能直接发给朋友。
结束送你一张「**看似贬低、实则封神**」的茶艺段位鉴定书，可一键导出 PNG 发朋友圈。

> ⚠️ 纯属娱乐 + 讽刺梗。玩的是「情商/拿捏/绿茶」这套网络话术的黑色幽默，别当真。

---

## 🎮 一局怎么玩

1. **开局随机三件套**：你的人设、今晚的富哥（大哥）、场地。
2. **10 轮对话**：每轮一段情境 + 3 个选项，选项对应不同「行为」。
3. **即时反馈**：对方反应一句、飘几条评论区弹幕、四条数值条实时跳动，连对触发 COMBO。
4. **结算鉴定书**：按最终「上头值」定档，再按你的画像抽一个具体结局；可导出图片、复制战绩、进图鉴收藏。

一句话目标：**读懂这个大哥的口味，投其所好，把「上头值」拉满，拿大结果（房 / 婚 / 出国 / 包养）。**

## 🧬 核心机制：三维口味匹配

### 你的人设 = A–F 六列各抽一词拼成（F 必出，其余按概率）

| 列 | 含义 | 偏「高级(+)」 | 偏「土甜(−)」 |
|---|---|---|---|
| A | 感觉 | 纯欲 / 清冷 / 氛围 / 白月光 / 妈生感… | 土纯 / 土甜 / 县城感 / 糙感美… |
| B | 面容类型 | 淡颜 / 浓颜 / 猫系 / 狐系 / 幼态 / 港风… | *（中性，只做风味）* |
| C | 气味 | 香香的 | 臭臭的 |
| D | 身高 | 175 / 170 | 165 / 160 |
| E | 学历 / 学科 | 文化艺术 / 时尚 / 网红 / 金融 / 法律 | 数学 / 计算机 / 大专 / 体育生 |
| **F** | **人设类型（必出）** | 富家女 / 千金 / 天菜 / 顶美 / 冷感美女 | 萝莉 / 甜妹 / 钓系 / 笨蛋美女 / 小美 |

六列的 +/− 累加，就是你这局固定的 **魅力值**（土甜 ◄──► 高级）。所以每局人设都不同，像「170·纯欲·猫系·金融·香香的·冷感美女」这种组合，抽卡感十足。

### 三条可正可负的维度

| 维度 | 一句话 | 负极 ◄──► 正极 | 谁决定 |
|---|---|---|---|
| **魅力** | 你是什么味的美 | 土甜 ◄──► 高级 | 人设六列（开局固定） |
| **框架** | 你有多拿得住 | 倒贴 ◄──► 拿捏 | 每轮行为 |
| **务实** | 你为爱还是为钱 | 纯情 ◄──► 搞钱 | 每轮「搞钱/不图钱」的抉择 |

### 上头值（主反馈条，0–100）

每一步你都在往某个 (魅力, 框架, 务实) 方向走。**大哥各有偏爱的象限**，你越贴近他要的方向，「上头值」涨得越猛；踩反了就掉。最终上头值决定结局大档：

| 上头值 | 结局档 |
|---|---|
| ≥ 70 | 🟣 大结果 |
| 45–70 | 🟢 小结果 |
| 20–45 | ⚪ 没结果·白上桌 |
| < 20 | 🔴 负结果 |

## 🎭 你每轮能打的牌（行为）

| 行为 | 走向 |
|---|---|
| 拿捏 / 掌握节奏 / 松弛 | 正框架（强势、掌控、松弛高价值） |
| 装傻 / 提供情绪价值 | 负框架（示弱、给予、哄） |
| 搞钱 | 正务实（要礼物、要好处、明码标价） |
| 不图钱 | 负务实（清高、不在乎他的钱） |

每轮都会**同时**给你「一个正框架 + 一个负框架 + 一个务实」的选项，所以无论对面吃哪套，你都有好牌可打——关键是你**读不读得懂他**。
台词还会**随人设口吻变化**：抽到「笨蛋美女」装傻就是「诶？人家没听懂啦～」，抽到「千金」不图钱就是「这点钱我爸给我当零花都嫌少」。

还有**彩蛋早退**：破绽爆表当场「翻车社死」，一眼万年提前「反向拿捏」，不用打满十轮。

## 🧠 一点策略

- **先读大哥**：他自我介绍偏"高冷/名媛"就要 +魅力 +框架；偏"实在/社恐"就 −框架、别搞钱。
- **人设是你的底牌**：抽到高级人设（+魅力）天生讨"高冷型"大哥喜欢；土甜人设去撩"实在型"更顺。魅力不匹配的大哥，再会打也可能只拿小结果——这就是抽卡的意义。
- **框架 vs 务实分开想**：他要强框架你就一路「拿捏」；他吃搞钱你就该要就要，别端着。
- **别贪也别倒贴**：破绽（负框架 + 乱搞钱）拉满 → 反被薅 / 翻车社死。

## 🖥️ 技术

- 单文件 `index.html`，原生 HTML/CSS/JS，**零外部依赖、零构建、可离线**。
- 移动端竖屏优先，安全区适配，深色霓虹风。

- 进度 / 图鉴用 `localStorage` 存本地。

## 本地玩
双击 `index.html` 即可，无需服务器、无需联网。

## 部署（Cloudflare Pages · 纯静态）
Cloudflare 控制台 → Workers & Pages → Create → **Pages → Connect to Git** → 选本仓库：
Framework preset = **None**、Build command **留空**、Build output directory = **`.`** → Deploy。
纯静态站、无后端、无构建步骤，Pages 直接发布 `index.html`；以后 `git push` 自动重新部署。

> 注意：要用 **Pages** 流程（只有 Build command + 输出目录），别用 Workers 的「Import a repository」（那个会跑 `wrangler deploy` 报错）。
